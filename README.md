# 西ソフト コーポレートサイト

西ソフト株式会社のコーポレートサイト。ラボ型開発・技術コンサルティングの事業内容を伝えるLP構成。

## 技術スタック

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4

## ブランド定義 v1.0

### カラートークン

| トークン | 値 | 用途 |
|---------|-----|------|
| `--brand-primary` | `#4666D6` | 見出しの一部 / 下線 / 強調語（1文1箇所まで） |
| `--brand-accent` | `#D34843` | CTAボタンのみ（1ページ最大1〜2箇所） |
| `--base-white` | `#FFFFFF` | 主背景 |
| `--base-gray` | `#D9D1CD` | セクション区切り背景のみ |
| `--base-black` | `#000000` | 本文・見出し・ロゴ |

### デザイン思想

- 静か、説明可能、制約が多い、装飾しない
- アイコン多用禁止、グラデーション禁止
- アニメーション最小限
- 最大幅 1100px、余白広め

### タイポグラフィ

- 日本語: Noto Sans JP
- 英字: Inter
- ウェイト: Regular (400) / Medium (500) / SemiBold (600) / Bold (700: Heroのみ)

## 現在の実装状況

### セクション構成（8セクション）

1. **Hero** - 主メッセージとサブテキスト
2. **Why** - 開発がうまく進まなくなる理由
3. **How** - 西ソフトのアプローチ
4. **Services** - ラボ型開発・技術コンサルティング
5. **Security** - セキュリティについて
6. **Team** - 経営陣（西方聖一、西川哲郎）
7. **Fit / Not Fit** - 向いているケース・向いていないケース
8. **Contact** - お問い合わせ（CTA）

### UIコンポーネント

| コンポーネント | 用途 |
|--------------|------|
| `Section` | セクションコンテナ（背景色・最大幅・余白） |
| `Heading` | 見出し（レベル1-3） |
| `Paragraph` | 本文 |
| `List` | 箇条書き |
| `Button` | CTAボタン（brand-accent） |
| `Divider` | 区切り線 |
| `Header` | ヘッダー（ロゴ + ダークモードトグル） |
| `Footer` | フッター（ロゴ + コピーライト） |

## 差し替え可能な箇所

### Team セクション

`components/sections/Team.tsx` の `members` 配列を編集。

### Contact セクション

`components/sections/Contact.tsx` の `formUrl` 変数を Google Form の URL に差し替え。

```typescript
const formUrl = "https://forms.gle/xxxx"; // ← 実際のURLに差し替え
```
